Подробное описание

Управляет SQLite-базой данных пользователей через Qt SQL.

Конструктор открывает БД и создаёт необходимые таблицы (если их нет). Деструктор закрывает соединение. Все публичные методы работают синхронно в вызывающем потоке.

Схема таблицы users:

id INTEGER PRIMARY KEY AUTOINCREMENT
name TEXT NOT NULL
login TEXT NOT NULL UNIQUE
hash TEXT NOT NULL -- MD5-хэш пароля
email TEXT NOT NULL
Исключения
std::runtime_errorпри ошибке открытия БД или инициализации таблиц.
