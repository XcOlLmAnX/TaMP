◆ parseRequest()

Request parseRequest ( const string & raw)

Разбирает сырую строку запроса в структуру Request.

Обрезает завершающие '\r'/'\n', делит строку по '&', сопоставляет первый токен с известными командами и проверяет количество аргументов. Несоответствие числа аргументов оставляет type == UNKNOWN.

Аргументы
rawСырая строка от клиента.
Возвращает
Структура Request с заполненными type и args.

Перекрестные ссылки Request::args, AUTH, CHECK, CONFIRM, NEWPASS, REG, RESET, split(), STAT, Request::type и UNKNOWN.

Используется в ServerInterface::handleClient().
