Описание проекта

TaMP — клиент-серверное приложение на C++/Qt, реализующее:

  • систему аутентификации (регистрация, вход, восстановление пароля через email);
  • интерактивное исследование кусочно-заданной математической функции с графиком и таблицей.

Архитектура

Проект разделён на два независимых подпроекта:

Подпроект Описание
TaMP_Client Qt Widgets GUI, TCP-клиент (QTcpSocket, Singleton)
TaMP_Server Консольное Qt-приложение, TCP-сервер (Winsock2)

Клиентская часть (TaMP_Client)

  • TcpClient — синглтон-обёртка над QTcpSocket, единая точка сетевого ввода-вывода.
  • LoginWindow — форма авторизации.
  • RegisterWindow — форма регистрации с клиентской валидацией.
  • ResetPasswordWindow — трёхшаговый мастер восстановления пароля.
  • TaskWindow — главное окно: интерактивный QCustomPlot-график и таблица значений.

Серверная часть (TaMP_Server)

  • ServerInterface — TCP-сервер на Winsock2, однопоточный accept-loop.
  • ServerLogic — диспетчер команд; маршрутизирует запросы к обработчикам.
  • DbHandler — RAII-обёртка над Qt SQL/SQLite для хранения пользователей.
  • SmtpClient — отправка transactional email через Mail.ru SMTP/SSL.
  • protocol.h — описание текстового протокола и функции make
  • Создано системой 1.16.1
