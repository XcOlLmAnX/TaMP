◆ doSend()

bool doSend ( const QString & toEmail,
const QString & subjectText,
const QString & bodyText )
static

Выполняет полный цикл SMTP-аутентификации и отправки одного письма.

Последовательность команд: SSL-handshake → EHLO → AUTH LOGIN → base64(login) → base64(password) → MAIL FROM → RCPT TO → DATA → QUIT. Тема письма кодируется в Base64 UTF-8 (RFC 2047, формат =?UTF-8?B?...?=).

Аргументы
toEmailEmail получателя.
subjectTextТема письма в UTF-8 (будет закодирована в Base64).
bodyTextТело письма в UTF-8 (plain text).
Возвращает
true при получении кода 250 после DATA, false при любой ошибке.

Ищет файл email.txt по трём путям и возвращает значение по ключу.

Пути поиска (в порядке приоритета):

  1. applicationDir/email.txt
  2. applicationDir/../email.txt
  3. currentDir/email.txt
Аргументы
keyКлюч для поиска в файле (например, "email" или "key").
Возвращает
Значение после символа '=' с обрезанными пробелами, или пустая строка.

◆ readResponse()

QString readResponse ( QSslSocket * socket,
int timeoutMs = 15000 )
static

Читает ответ SMTP-сервера и возвращает последнюю строку.

Многострочные ответы SMTP (например, после EHLO) усекаются до последней строки, которая содержит финальный код ответа.

Аргументы
socketSSL-сокет для чтения.
timeoutMsТаймаут ожидания в миллисекундах (по умолчанию 15 с).
Возвращает
Последняя строка ответа сервера, или пустая строка при таймауте.
