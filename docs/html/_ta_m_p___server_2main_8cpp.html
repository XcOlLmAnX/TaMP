Подробное описание

Точка входа серверного приложения TaMP.

Инициализирует три ключевых компонента:

  • DbHandler — работа с SQLite-базой данных пользователей;
  • ServerLogic — бизнес-логика обработки запросов;
  • ServerInterface — TCP-сервер на базе Winsock2.

При ошибке инициализации любого компонента в stderr выводится сообщение и возвращается код завершения 1.

Функции

◆ main()

int main ( int argc,
char * argv[] )

Запускает серверное приложение TaMP.

Создаёт QCoreApplication (необходим для Qt SQL и SSL), затем последовательно инициализирует DbHandler, ServerLogic и ServerInterface. ServerInterface::run() блокирует поток и обрабатывает клиентов последовательно (один за другим без многопоточности).

Аргументы
argcКоличество аргументов командной строки.
argvМассив строк аргументов командной строки.
Возвращает
0 при нормальном завершении, 1 при исключении инициализации.
