Подробное описание

Единственный объект TCP-соединения в клиентском приложении.

Реализует паттерн Singleton: конструктор приватный, копирование и присваивание запрещены, единственный экземпляр доступен через getInstance(). Все окна приложения разделяют один сокет и получают входящие данные через сигнал messageReceived(). Данные читаются построчно — одна строка = один ответ сервера.

Заметки
Потокобезопасность не гарантируется; весь ввод-вывод происходит в главном потоке Qt.
