Подробное описание

Форма входа в систему с полями логина и пароля.

Отправляет серверу запрос вида auth&{login}&{md5(password)}\n и обрабатывает ответы с префиксами auth+ (успех) и auth- (ошибка). При успешном входе испускает loginSuccess() с именем пользователя.

◆ onLoginClicked

void LoginWindow::onLoginClicked ( )
privateslot

Валидирует поля, хэширует пароль и отправляет запрос авторизации серверу.

Проверяет заполненность полей, MD5-хэширует пароль и отправляет запрос серверу.

Формат запроса: auth&{login}&{md5hash}\n. При незаполненных полях выводит сообщение об ошибке и устанавливает фокус на поле.
