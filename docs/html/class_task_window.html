Подробное описание

Интерфейс для исследования кусочно-заданной функции с параметрами a, b, c.

Функция определена как:

f(x) = { -x + a, x < 0
{ x^b, 0 <= x < 2
{ x/2 + c, x >= 2
TaskWindow::f
double f(double x, int a, int b, int c)
Вычисляет значение кусочной функции в точке x.
Определения taskwindow.cpp:161

Три слайдера (a, b, c ∈ [−10, 10]) управляют параметрами в реальном времени. QCustomPlot отображает три цветовых сегмента функции; QTableWidget — таблицу значений для x ∈ {−10, −8, …, 10}.

◆ setupPlot()

void TaskWindow::setupPlot ( )
private

Применяет стиль к QCustomPlot: тёмный фон, белые оси, сетка, легенда.

Настраивает внешний вид QCustomPlot: цвета, оси, три графика, легенду.

Каждый из трёх графиков (graph 0/1/2) соответствует одной ветви функции: синий — x < 0, зелёный — 0 ≤ x < 2, красный — x ≥ 2. Включены зум и перетаскивание (QCP::iRangeDrag | QCP::iRangeZoom).

◆ updatePlot

void TaskWindow::updatePlot ( )
privateslot

Пересчитывает и отрисовывает график и таблицу по текущим значениям слайдеров.

Считывает значения слайдеров, пересчитывает три ветви функции и перерисовывает график.

Для каждой ветви функции генерируется набор из N=500 точек на соответствующем интервале. Данные передаются в графики 0, 1, 2 QCustomPlot. После перерисовки вызывается updateTable().
